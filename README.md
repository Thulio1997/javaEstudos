# Java Estudos

Repositório com exercícios e projetos simples em Java para praticar lógica de programação, entrada de dados, cálculos e estrutura básica da linguagem.

## Projetos

### Hello World

Primeiro programa em Java utilizando `System.out.println().

### Calculadora de Desconto

Programa que calcula desconto de 15% para produtos acima de R$200.

### Calculadora de IMC

Programa que calcula o IMC com base no peso e altura informados pelo usuário.

### Leitura de Números Inteiros

Programa que lê dois números inteiros usando `Scanner` e exibe os valores digitados.

## Tecnologias

* Java
* Scanner
* VS Code

## Aprendizados

* Estrutura básica do Java
* Entrada de dados
* Variáveis
* Condicionais (`if/else`)
* Operações matemáticas
* Uso do `Scanner`
* Compilação e execução na JVM
